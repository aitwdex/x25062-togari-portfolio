# x25062-togari-portfolio
Webデザインおよび演習のリポジトリ作成！

機能説明
・Aboutの顔写真をクリックで変更可能
・Worksの写真や映像を矢印や下の5つの丸から変更可能
・Worksの映像作品をクリックで再生可能
・Contactの文字入力可能、選択可能、送信可能
　（実際にメールに送られるようになっている。）
・snsアイコンから新しいタブでsnsに飛ぶことができる
・作品一覧btnからWorksの別パージに飛べる
・navのボタンから別のページに飛べる
・右下の上矢印btnからtopに戻れる
・画面サイズが小さいとハンバーガーが出てくる(操作可能)

・タグをクリックで種別に作品を見せる
・クリックで動画再生
・再生可能な作品ホバー時に枠線の色変化


工夫点
・formのfieldsetなどで読み上げ機能などに対応するようにした。
・topに戻るボタンのように便利な機能も付属で追加した

・スマホサイズにも対応
